About

This simulator uses Monte Carlo methods to price European call and put options, comparing results with the Black-Scholes closed-form solution.

Mathematical Foundation

  • Geometric Brownian Motion: S_{t+1} = S_t × exp((r - 0.5σ²)dt + σ√dt×Z)
  • Call Option: C = e^(-rT) × E[max(S_T - K, 0)]
  • Put Option: P = e^(-rT) × E[max(K - S_T, 0)]

Features

  • High-precision Monte Carlo simulation
  • Statistical error analysis
  • Real-time performance metrics
  • Risk-neutral pricing framework
